## 已确认决策

- 纯前端静态工具。
- 不需要后端和服务器 API。
- 混合输入：支持整段 AI 回复和单条命令。
- 自动优先。
- 支持多条命令分别复制和复制全部。
- 本地存储只保存偏好和自定义规则。
- 默认不保存命令历史。
- here-doc 第一版不处理。
- 核心解析采用状态机规则引擎。

## Agent 建议摘要

- 设计：采用“命令修复工作台”而不是营销页；桌面端左输入右结果，移动端单列；命令内容横向滚动；复制反馈清晰。
- 测试：P0 必须覆盖 Python `-c` SQL 折行、多命令识别、复制全部、反斜杠续行、JSON 参数、here-doc 不支持、高风险提示、本地存储不保存历史。
- 运维：MVP 推荐 Vercel 或 Cloudflare Pages；仍保持纯静态可移植；不要把命令写入 URL；发布前检查无 API、无数据库、无历史保存。

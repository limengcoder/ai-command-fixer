## 已确认决策

- 纯前端静态工具。
- 不需要后端和服务器 API。
- 输入默认按一条长命令处理。
- 自动优先。
- 支持单条复制按钮和点击结果框复制。
- 支持用户主动本地暂存修复后的完整命令；默认不自动保存。
- 多命令识别后置为显式高级模式，不作为 MVP 默认体验。
- 本地存储只保存偏好和自定义规则。
- 默认不保存命令历史。
- here-doc 第一版不处理。
- 核心解析采用状态机规则引擎。
- 开源前允许隐私友好的匿名统计接入，并支持 GSC、GA 等主流统计指标配置。
- 后续开源选型由产品经理定义标准，开源选型 agent 调研，前端 agent 做技术评估，产品经理汇总建议。
